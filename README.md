simple console game

#### Prerequisites:
- install [Maven](https://maven.apache.org/download.cgi). 

#### How to start up ?
- git clone
- build the maven project ```mvn package```
- start app with the below cmd
          
            java -cp target/D-D-RPG-1.0-SNAPSHOT.jar io.rybak.StartGame
